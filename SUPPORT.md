# Support

## Where to Ask for Help

- **Usage questions / integration help:** open a GitHub Discussion (preferred).
- **Bug reports / feature requests:** open a GitHub Issue.
- **Security issues:** follow [SECURITY.md](./SECURITY.md) and report privately.

## Before Opening an Issue

Please include:

- Node.js version
- Package version (`webrtc-rooms`)
- OS and environment details
- Minimal reproducible example
- Expected behavior vs actual behavior
- Relevant logs and stack traces

## Issue Quality Checklist

- Reproducible with current `main` or latest npm release
- Clear title and scope
- One issue per problem
- No secrets or private keys in logs

## Maintenance and Response

Maintainers aim to:

- Respond to new issues within **7 days**
- Label and triage by severity/priority
- Close stale issues after follow-up when no reproduction is provided
